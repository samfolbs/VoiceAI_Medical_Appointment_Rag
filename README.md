# Medical Triage Voice Assistant with RAG

> **FastAPI  · Deepgram · ChromaDB · Python 3.11+**

A real-time, voice-first scheduling system that lets patients book medical appointments through natural spoken conversation. It combines speech-to-text, a LLM reasoning layer, retrieval-augmented generation (RAG) over a medical knowledge base, and text-to-speech — all delivered over a WebSocket connection from a browser.

>  **Emergency Safety** — The assistant detects life-threatening symptoms (such as chest pain, stroke, severe bleeding) before any other action and immediately directs the patient to call 911 or go to the nearest emergency room. No appointment is ever scheduled for an emergency condition.
